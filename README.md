# projects-from-course
Just repo for some projects

### Executing the program
Clone the repository:
```
git clone https://github.com/Lirmortoch/projects-from-course.git
```
Select the folder you want (example bellow):
```
cd "folder-name"
```
Install dependencies:
```
npm install
```
Run the server:
```
npm run dev
```
or 
```
npm run build
npm run preview
```
